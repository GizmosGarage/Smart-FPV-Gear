Daily Logs — Smart FPV Gear

Purpose
This folder contains day-to-day development and experimentation logs for the Smart FPV Gear project.

How to add an entry
- Create a new Markdown file named with the date and short title, for example:
  2025-12-11-flight-test.md
- Include a brief header, summary, and any links or images.

Suggested front matter (optional):
---
date: 2025-12-11
tags: [testing, flight, notes]
---

Entry template
# 2025-12-11 — Flight Test
Summary:
- Short summary here.

Details:
- What I tried
- Observed results
- Artifacts (logs, images, links)
- Next steps

Notes
- Keep entries concise and focused.
- Store large images or binary artifacts elsewhere (or in daily-logs/images/) and link to them.
- Use tags in front matter to help filtering and searching later.

If you'd like, I can also open a pull request from branch add-daily-logs-folder instead of pushing to main.
